Тарификация: Мобильная связь - приложение, предназначенное для просмотра информации по продолжительности и стоимости голосовых вызовов, количества и стоимости мобильного интернета, а также других услуг, оказываемых операторами по корпоративным договорам мобильной связи.

Сотрудник компании видит данные по тарификации мобильной связи:

  • обобщенные данные по SIM-картам, которые выданы непосредственно сотруднику, так же сотрудник имеет возможность посмотра детализированной информации по осуществленным вызовам и оказаным услугам на этих SIM-картах.
  • обобщенные данные по SIM-картам, которые были выданы коллегам(сотрудники компании, работающие в том же подразделении) сотрудника. Сотрудник не имеет права просмотра детализированной информации по SIM-картам коллег.
  • обобщенные и детализированные данные по SIM-картам, которые были выданы подчиненным сотрудникам.
  • обобщенные и детализированные данные по всем SIM-картам, если сотрудник является администратором/наблюдателем тарифов связи.
Основы работы с приложением:

Поля фильтрации:
  • Незакрытый месяц - предварительные данные за прошлый и текущий месяц.
  • Год - список формируется от первого года начала загрузки данных тарификации.
  • Месяц - полный список месяцев, не зависит от доступных данных тарификации.
  • Договор - договора, по которым получены SIM-карты, тарификация которых доступна сотруднику.
  • Сотрудник - список сотрудников: собственно сотрудник, коллеги и подчиненные, тарификация по SIM-картам которых доступна сотруднику.
  • Телефон - список телефонных номеров SIM-карт, которые доступны сотруднику.
Данные в элементах фильтра зависят от значений друг друга. Установлена следующая последовательность зависимостей:
Год -> Месяц -> Договор -> Сотрудник -> Телефон
Для того, чтобы посмотреть данные тарификации, удовлетворяющие условиям фильтрации нажмите на Искать

Управляюшие элементы:
  • Очистить фильтр - при нажатии очищает установленные критерии фильтрации и таблицу с данными тарификации.
  • Искать - при нажатии осуществляется поиск и вывод данных тарификации, удовлетворяющих условиям фильтрации.

Таблица с данными тарификации:
Данные в таблице группируются по трем уровням:
  • Договора сотовой связи
  • Сотрудники
  • Телефонные номера - на данном уровне, при наличии прав, можно открыть детализированный отчет по тарификации конкретного телефонного номера.
Если в поле месяц выбрано условие "весь год (без детализации)" порядок группировки изменяется на:
  • Месяца
  • Договора сотовой связи
  • Сотрудники
В таблице предусмотрено цветовое деление колонок "Сумма" и "Сумма, оплачиваемая сотрудником" в зависимости от способа начисления НДС в соответствии с условиями договора сотовой связи. Способы начисления НДС и соотвествующие им цвета описаны в легенде под таблицей с данными тарификации.

Отчеты:
  • Затраты по телефонам - данные по телефонным номерам, по которым по данным тарификации необходимо заплатить наибольшие суммы
  • Длительность по телефонам - данные по телефонным номерам, по которым осуществлено больше всего вызовов
  • Затраты по сотрудникам - данные по сотрудникам, по которым по данным тарификации необходимо заплатить наибольшие суммы
  • Длительность по сотрудникам - данные по сотрудникам, которые больше всех осуществили вызовов
- все отчеты строятся в зависимости от установленного фильтра.
